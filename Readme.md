# 🌐 Portafolio Web Personal

Bienvenido a este repositorio. Este proyecto corresponde a mi portafolio web personal, desarrollado utilizando **HTML, CSS y JavaScript**. Su propósito es presentar de manera profesional información sobre mí, mis habilidades, proyectos y experiencia en el desarrollo de software.

He decidido publicar este proyecto de forma abierta para que cualquier persona pueda utilizarlo como referencia, aprender de su estructura o adaptarlo a sus propias necesidades.

## 📖 Descripción

Este portafolio fue construido utilizando tecnologías web fundamentales, priorizando la simplicidad, el rendimiento y la facilidad de personalización. No requiere frameworks ni dependencias externas complejas, por lo que puede ser modificado fácilmente por cualquier desarrollador con conocimientos básicos de desarrollo web.

El diseño de este proyecto fue inspirado y tomado como referencia de diversos **templates disponibles en Framer**, los cuales sirvieron como base para la estructura visual, la distribución de componentes y la experiencia de usuario. Posteriormente, la implementación fue desarrollada manualmente utilizando HTML, CSS y JavaScript.

## 🚀 Características

- Diseño moderno y responsivo.
- Compatible con dispositivos móviles y de escritorio.
- Código limpio y fácil de entender.
- Secciones completamente personalizables.
- Fácil integración de nuevos proyectos y contenido.
- Implementado únicamente con HTML, CSS y JavaScript.
- Inspirado en diseños y plantillas de Framer.

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript

## 📂 Instalación y Uso

1. Clona este repositorio:

```bash
git clone https://github.com/AdderlyMendoza/Portafolio-2026.git
```

2. Accede al directorio del proyecto:

```bash
cd portafolio-2026
```

3. Abre el archivo `index.html` en tu navegador.

También puedes utilizar herramientas como **Live Server** en Visual Studio Code para ejecutar el proyecto localmente.
